**Mongoose and Forms Quiz:**




**1. What is the method for getting all the documents in the database for the model?.**

**a.** .find( )



**2. What is the method for creating a new document in the database for the model?.**

**a.** .create( )



**3. What is the method to get 1 document by it’s id?.**

**a.** .findById( )



**4. What is the method for getting a document by its id and updating it?.**

**a.** .findByIdAndUpdate( )



**5. What is the method for getting a document by its id and deleting it?.**

**a.** .findByIdAndDelete( )



**6. What are the 2 arguments of the findByIdAndUpdate() method?.**

**a.** the id and the updated data object



**7. What data type is req.params.**

**a.** object { }



**8. Let’s say you have a route like this app.get(‘/fruits/:id’). How can you access the dynamic route parameter in the function.**

**a.**  req.params.id



**9. What do each attribute for the form do: method and action?.**

**a.** method is get or post, the action is where the data is sent
